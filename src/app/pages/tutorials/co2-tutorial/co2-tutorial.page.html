Le Fonctionnement Comment le nombre de pas se calcule ? Toute les applications qui ont ce module(pédometre) utilise le device motion, qui est un capteur de mouvement du smartphone, donc dés que vous commencer à marche(faire des mouvementsavec vos mains) le nombre de pas s'incrémente ainsi que les autres données(CO2, calories et distance). pour que le nombre de pas ne s'icrémente plus, vous devez placer votre téléphone sur table par exemple et vous allez voir que le nombre de pas ne s'incrémente plus.
